== INSERÇÃO DE FORMULÁRIOS ==

<input type="text">  formulario de texto --> se quiser colocar algum texto padrão, add value"" ( exemplo:  <input type="text" value"@gmail.com"> no formulário já estará o @gmail.com
<input type="password"> formulario de senha
<input type="checkbox"> checkbox ( marcador de vários itens)
<input type="radio" name="sexo"> marcador, porém marca uma ou outra ( exem: sexo masculino ou fem)
<input type="submit"> - tag para envio do formulário ( por padrão o botão estará escrito enviar) para alterar add value:""
<input type="email"> - tag igual text porém obriga a colocar um email 

<input type="datetime-local" name="usuario"> - data e hora
<input type="date" name="usuario"> - data
<input type="color" name="usuario"> - cor
<input type="month" name="usuario"> - mes
<input type="number" name="usuario"> - só aceita numeral
<input type="range" name="usuario"> - abre uma barrinha ----o--
<input type="search" name="usuario"> - igual ao text
<input type="tel" name="usuario"> - telefone
<input type="time" name="usuario"> - tempo
<input type="week" name="usuario"> - envia a semana do ano

  <select name="ordenacao">
            <option> mais vendidos </option>
            <option> mais recentes </option>
            <option> marbarato </option>
         </select>   
----> abre uma aba com essas opções 
  <select name="ordenacao">
            <option value"bestseller"> mais vendidos </option>
            <option value"last"> mais recentes </option>
            <option> marbarato </option>
         </select>   
----> abre uma aba com essas opções e com value, envia o valor do value ( na página do user)


         <select name="order">
            <option value="bestseller"> mais vendidos </option>
            <option value="last" selected> mais recentes </option>
            <option> marbarato </option>
         </select>   

----> adicionando o selected na tag, a página já abre com o pre selecionado da tag, no caso vai aparecer o "mais recentes"


-----
<select name="order" size="2" multiple>
            <option value="bestseller"> mais vendidos </option>
            <option value="last" selected> mais recentes </option>
            <option> marbarato </option>
         </select>   
---> add o size e multiple, você deixa o usuario selecionar mais de uma option
